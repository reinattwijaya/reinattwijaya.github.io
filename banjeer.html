Banjeer

This is a website for a hackathon, we made this website for our Machine Learning API. The idea is simple: we create an API that takes image of a river as an input and output the quality of the river as an output. This is really crucial in today's society especially to be able to monitor river's quality and provide support efficiently without going there. The benefits include:

  • Government work: managing the rivers around the city and making sure that each river is not on the verge of flooding. This would be a really great way to save money and time rather than doing monthly checking all the time
  • Society: Provide the awareness of our current river quality for the society to see. With this webiste, we hope that now any regular people do not need to be an expert in water or river quality to understand our current environment situation
  • Develop a more established model and web so that we can expand this feature to other things such as sea or lake
This website is dedicated for the Hackathon Imagine Cup Microsoft and the purpose is to help bring a better world for all of us.
